Resources

This page contains pointers to some third-party resources, in particular distro repos for an easier installation of Janus, where available, client stacks, libraries and bindings in other languages, that is tools that you find helpful to interact with Janus outside of the context of the janus.js JavaScript library we provide. It is not a complete list, but just a summary of the material that has been shared so far on the meetecho-janus Google group.

If you've developed anything related to Janus (client stacks, plugins, transports, etc.) and you're willing to share them with the community, or you know any such effort that is not listed here, just let us know and we'll add it on this page.
